**Target Languages:** Go, Zig, C, C++, C#
**Core Guidelines:**
1. **Early Returns**
   Always use guard clauses / early returns to avoid nested ifs. Keep code flat.
2. **No Magic Values**
   Extract every string literal and magic number into named const/var at top of function/scope.
3. **Line Length**
   Ideal under 80 characters.
